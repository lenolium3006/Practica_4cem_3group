# LibBot

LibBot — Telegram-бот, разработанный для демонстрации контейнеризированного развертывания приложения с использованием Docker и PostgreSQL.

## Запуск проекта

Сборка Docker-образа:

```bash
docker compose build
```

Запуск контейнеров:

```bash
docker compose up
```

Проверка состояния контейнеров:

```bash
docker ps
```

Остановка приложения:

```bash
docker compose down
```

## Структура проекта

* `echobot.py` — основной файл Telegram-бота;
* `Dockerfile` — описание сборки Docker-образа;
* `docker-compose.yml` — конфигурация запуска контейнеров;
* `.env` — переменные окружения;
* `requirements.txt` — зависимости проекта.

## Результат

После запуска бот принимает сообщения от пользователей Telegram и отправляет их обратно (режим эхо-бота).
